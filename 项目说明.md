# FastAPI 全栈脚手架

基于 [FastAPI Full Stack Template](https://github.com/fastapi/full-stack-fastapi-template) 的前后端分离脚手架，用于快速启动新项目。

## 技术栈

- **后端**：FastAPI、SQLModel、PostgreSQL、Alembic、JWT 认证
- **前端**：React、TypeScript、Vite、TanStack Router/Query、Tailwind、shadcn/ui
- **部署**：Docker Compose、Traefik（生产）、Nginx（前端静态资源）

## 功能

- 用户注册 / 登录 / 找回密码
- JWT 鉴权与超级管理员用户管理
- **RBAC**：权限码、角色、用户角色分配（`/admin/permissions`）
- **动态菜单**：按角色控制侧栏与路由访问（`/admin/menus`）
- Items 示例 CRUD（演示前后端协作）
- OpenAPI 自动生成前端 API 客户端
- 中英文界面（i18next）
- 本地 Mailcatcher 邮件调试
- Playwright E2E 测试

## 快速开始

```bash
# 后端与数据库（Docker）
docker compose up -d db backend adminer mailcatcher

# 前端（本地，支持热更新）
cd frontend
bun install   # 首次
bun run dev
```

浏览器打开 http://localhost:5173 。后端 API：http://localhost:8000/docs 。

详见 [development.md](./development.md) 与 [访问地址.md](./访问地址.md)。

## 生成前端 API 客户端

后端 OpenAPI 变更后执行：

```bash
./scripts/generate-client.sh
```

## 目录说明

| 路径 | 说明 |
|------|------|
| `backend/app` | FastAPI 应用与 API |
| `frontend/src` | React 前端 |
| `compose.yml` | 生产/通用 Compose 配置 |
| `compose.override.yml` | 本地开发覆盖（端口、热重载等） |

## 启动

```bash
# 重建数据库（仅首次或迁移变更后）
docker compose down -v
docker compose up -d db backend

# 前端本地开发
cd frontend && bun run dev
```
